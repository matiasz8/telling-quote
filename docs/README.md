# Documentation Hub

## Objetivo

Estandarizar la documentacion de telling-quote con la misma estructura operativa de frontend-template.

## Estructura oficial

- `00-roadmap/`: fases, estado y guias de navegacion.
- `01-governance/`: workflow para crear y mantener PRD/TRD/ARD.
- `02-prd/`: decisiones de producto (que se construye y para quien).
- `03-trd/`: decisiones tecnicas (como se implementa).
- `04-ard/`: decisiones arquitectonicas de largo plazo.

## Regla obligatoria para nuevos proyectos y features

Todo trabajo nuevo debe incluir:

1. `PRD`.
2. `TRD`.
3. `ARD`.

Sin estos tres documentos, el trabajo se considera incompleto.

## Ruta de lectura recomendada

1. `00-roadmap/where-to-find-what.md`.
2. `00-roadmap/phases.md`.
3. `01-governance/governance-workflow.md`.
4. Documentos del feature en `02-prd/`, `03-trd/`, `04-ard/`.

## Estado de migracion

- PRDs migrados desde `docs/prd/` a `docs/02-prd/`.
- TRDs migrados desde `docs/trd/` a `docs/03-trd/`.
- Estructura 00-04 creada y activa.
