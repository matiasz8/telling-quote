## Como usar esta carpeta

1. Leer el PRD de la funcionalidad a modificar.
2. Validar objetivos, no-objetivos y criterios de aceptacion.
3. Confirmar trazabilidad hacia TRD y ARD.

## Regla obligatoria

Todo PRD debe tener su TRD y ARD vinculados antes de cierre.
